# Stoat Chat

![stoat](stoat_chat.jpeg)

A simple chat system which based on websockets and public key authentication. Instead of the traditional method of keeping users and passwords


## Schema

Message schema
```json
{
  type: 'message',
  data: {
    text: 'message text',
    pubkey: 'meow',
    signature: 'meowmeow',
    nick: 'stoat-chan',
    id: 'meow',
    time: 'Mon Oct 26 04:47:09 UTC 2009'
  }
}
```
